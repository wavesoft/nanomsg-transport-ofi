\abstract{
The \ofi\ transport is a new transport to nanomsg that enables access to high-performance fabrics, such as usNIC, Infiniband, Omni-Path etc., using the libfabric library. By extending nanomsg we are perserving the Scalability Protocols provided by the library, thus greatly simplifying the integration with existing project.
}
